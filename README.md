
AR相机
===================================
学习GLSL语言的练习项目。
  
在大神的开源项目基础上扩展了一些滤镜。
  
另外，通过动态人脸模型还实现了类似Faceu的换脸效果。
  
项目中用到的滤镜、3D模型都不是那种打包加密的文件，可随意添加、修改。
  
[这篇博客分享开发思路](http://blog.csdn.net/torchlight2012/article/details/75722424)
